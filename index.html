My_Profile

Repositorio base para definir y mantener un perfil profesional reproducible con utilidades de Python y controles de QA/QC. Abre este archivo directamente o sirve el directorio con python -m http.server para ver esta vista.

Estado

Módulo my_profile.profile.Profile con validaciones + pruebas unitarias.

Flujo CI: ruff, black --check, pytest.

Uso rápido

Ejecuta el ejemplo:

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python - <<'PY'
from my_profile.profile import Profile
perfil = Profile(
    nombre="Ada Lovelace",
    rol="Pionera de la programación",
    habilidades=["Matemática", "Algoritmos", "Documentación"],
    ubicacion="Londres, Reino Unido",
)
print(perfil.resumen())
PY

Calidad

Comandos principales:

ruff check src tests
black --check src tests
pytest --cov=src --cov-report=term-missing --cov-fail-under=90

Si no se dispone de pytest-cov, usar pytest simple y reportar la limitación.
